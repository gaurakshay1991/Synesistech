# Synesis Private Investor Handover

## Positioning

Synesis is a private legal intelligence platform for live legal research, contract review, clause rewriting, legal memo generation and regulatory watch.

This build is for founder/investor demonstration only. It has no payment module and no public self-signup.

## Default private access

Change these values in your local environment file before showing the product to investors.

- Admin: admin@synesis.local
- Investor: investor@synesis.local
- Reviewer: reviewer@synesis.local

## Modules

1. Live Legal Research
2. Contract Review
3. Clause Rewrite
4. Legal Memo
5. Regulatory Watch
6. Admin Audit

## Security rules

- Keep repository private.
- Do not commit the real OpenAI API key.
- Put the key only in the backend environment.
- Change all demo passwords before any investor demo.
- Do not enable public signup.
- Do not upload confidential client documents until access controls, storage, deletion, audit and data handling are reviewed.

## Investor demo mode

Run locally or on a private deployment. Screen-share the product or give investor credentials only to selected persons.
