# THEMIS Product Suite

The `themis` branch is an independent institutional decision and execution platform. It preserves the existing Decision OS and Capital & Scenario Lab and adds functional prototype workspaces for:

- Institutional Decision Twin
- Regulatory Decision Assurance
- Contract Value Assurance
- Counterparty Event Command
- Decision Memory
- Governance & Authority
- AI Control Tower
- Agent Studio
- Evaluation Lab
- Integration Hub
- Evidence & Audit Command

## Current operating boundary

The new workspaces provide structured registers, workflow statuses, search, export and browser persistence. The institutional database migration, Render Blueprint and CircleCI configuration are prepared but require external service activation and credentials before central multi-user persistence, SSO, licensed data and production integrations become active.

## Product rule

No vertical product replaces the platform. Every product must converge on the same Institutional Decision Twin, Decision Memory, governed AI controls, execution model and evidence architecture.
