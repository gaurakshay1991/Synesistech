# Synesis Institutional Intelligence Prototype

This branch contains the working AMC and regulated-enterprise prototype. It is intentionally positioned as an institutional intelligence and execution platform, not as a legal-tech product.

The prototype performs fresh analysis of user-supplied documents and portfolio CSV data, extracts mandate restrictions, simulates redemption scenarios, maps regulatory changes to actions, preserves browser-local decision memory, and displays live configuration status. Demonstration records are clearly labelled and remain separate from uploaded analyses.
